# 极简朋友圈

> 本项目复刻自kingwrcy大佬的[moments](https://github.com/kingwrcy/moments)

## 项目介绍

已经实现的功能:

- 支持原生全部功能
- 自动获取位置信息
- markdown支持
- 无感验证码支持
- 消息推送（邮件）
- 多用户
- 上传完的图片可以拖动排序

## 项目介绍

已经实现的功能:

- 支持原生全部功能
- 自动获取位置信息
- markdown支持
- 无感验证码支持
- 消息推送（邮件）
- 多用户
- 上传完的图片可以拖动排序
- tag/标签功能
- @/提及功能
- 查看权限功能，支持私密，或者部分用户可见
